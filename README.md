# Todo List

Простое одностраничное приложение для управления задачами,  
реализованное на ванильном JavaScript. Демонстрирует базовые  
фронтенд-навыки без использования фреймворков.

## Структура проекта

```
todos/
├── src/
│   ├── js/
│   │   ├── constants.js            # Основные константы
│   │   ├── main.js                 # Главный JS-файл
│   │   ├── state.js                # Управление состоянием приложения (CRUD операции, работа с localStorage)
│   │   ├── theme.js                # Логика переключения светлой/тёмной темы
│   │   ├── todo-handlers.js        # Обработчики действий с задачами (добавление/удаление)
│   │   ├── todo-render.js          # Рендеринг интерфейса задач и фильтров
│   │   └── utils.js                # Вспомогательные функции
│   ├── scss/
│   │   ├── base/                   # Сброс стилей, mixins, переменные
│   │   ├── components/             # Стили компонентов
│   │   ├── utils/                  # Стили для тёмной темы
│   │   ├── _responsive.scss        # Адаптив
│   │   └── index.scss              # Главный файл стилей
│   └── index.html                  # Основной HTML
├── package.json
├── package-lock.json
├── webpack.config.js       # Конфигурация сборки
└── README.md
```

### Важные файлы

- `src/index.html` - HTML-страница
- `src/js/main.js` - иницилизирует приложение
- `src/scss/index.scss` - собирает всё SCSS
- `src/scss/base/_variables.scss` - палитра и переменные
- `webpack.config.js` — настроенная сборка, обработка SCSS/JS/asset-файлов

## Что было сделано

- Верстка страницы
- Настройка адаптивности
- Структурирование JS по модульному подхода
- Структурирование SCSS по методологии компонентного подхода
- Настройка Webpack для сборки (SCSS → CSS, минификация, копирование ассетов)

## Реализованный функционал

- Собранная адаптивная HTML-страница
- Корректная организация файловой структуры
- Обработка стилей SCSS через Webpack
- Поддержка переменных, миксинов и отдельных блоков
- Автоматическая сборка и запуск дев-сервера

JS функционал:

- Добавление, удаление и отметка выполнения задач
- Фильтрация: все, активные, выполненные
- Переключение светлой/тёмной темы
- Данные сохраняются в localStorage

## Используемые технологии

- `HTML5` / `CSS3` / `SCSS`
- `JavaScript (ES6+)`
- `Webpack 5`
- `File-loader` / `Asset modules`
- `Node.js` / `npm`

## Требования к запуску

- `Node.js` **16+**
- `npm` **8+**

### Инструкция

1. Клонировать репозиторий:
   `git clone https://github.com/MilanaMur/todo-list.git`
2. Перейти в папку проекта:
   `cd todo-list`
3. Установить зависимости:
   `npm install`
4. Запустить дев-сервер:
   `npm run start`
